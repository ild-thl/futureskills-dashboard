Bei einer Sentimentanalyse oder auch Stimmungsanalyse wird ein Text automatisiert nach geäußerten Gefühlen oder Meinungen ausgewertet. Ein Text wird anhand seiner Wörter klassifiziert und innerhalb einer Skala als positiv, negativ oder neutral eingestuft.
Anwendungsbeispiele finden sich in der Analyse von Literatur, für die Klassifizierung von Kommentaren, bspw. in Blogs, Social Media Anwendungen oder Produktbewertungen.
